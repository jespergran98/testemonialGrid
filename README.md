CSS Advanced Oppgave 1: Testimonial Grid

Du skal lage en "Testimonials"-seksjon med et responsivt gridoppsett. Seksjonen skal inneholde flere bokser med tilbakemeldinger fra brukere. Hver boks skal inneholde informasjon som brukernavn, bilde, yrke, og selve tilbakemeldingen.

Last ned filene til oppgaven her.Links to an external site.

Bruk CSS Grid til å lage oppsettet, og følg disse spesifikasjonene:

Krav til oppgaven:
HTML-struktur:

Lag en container <div> som holder hele testimonials-seksjonen.
Hver testimonial skal være en egen <div>.
Innholdet i hver testimonial skal inkludere:
Et bilde av brukeren.
Brukernavn.
Brukerens yrke/tittel.
Tekst med selve tilbakemeldingen.
Grid-oppsett:

Sett display: grid på containeren.
Bruk grid-template-columns og grid-template-rows for å definere kolonner og rader.
Du kan bruke grid-template-areas i stedet for grid-row og grid-column for mer avansert layout (valgfritt).
Bruk gap til å lage mellomrom mellom elementene.
Testimonials kan ha forskjellige størrelser eller strekke seg over flere kolonner/rader (bruk grid-column og grid-row).
Responsiv design:

Bruk media queries for å lage et responsivt oppsett:
For mobilvisning: Bruk 1 kolonne.
For skrivebord: Bruk 4 kolonner.
Stil:

Følg den vedlagte stilguiden for farger, skrifttyper osv.
Bruk padding og margin for å skape god avstand.
Legg til myke avrundede hjørner med border-radius.
Bruk box-shadow for å gi et subtilt 3D-utseende.
Noen styling-egenskaper, som bakgrunnsfarger eller overskrifter, bør gjøres gjenbrukbare ved å tildele dem klasser.
